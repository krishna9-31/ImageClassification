# ♻️ E-Waste Generation Classification

A Machine Learning project that classifies electronic waste (E-Waste) images into different categories using Deep Learning techniques. The objective of this project is to automate the identification of e-waste, enabling efficient waste segregation and supporting sustainable recycling practices.

The project covers the complete machine learning pipeline, including data preprocessing, image augmentation, model training, evaluation, and prediction using Convolutional Neural Networks (CNNs).

---

## 🚀 Features

- Image preprocessing and normalization
- Data augmentation for improved model performance
- Deep Learning-based image classification
- Model training and evaluation
- Prediction on unseen E-Waste images
- Training and validation accuracy visualization
- Easy-to-understand Jupyter Notebook implementation

## 📊 Dataset

The project uses an E-Waste image dataset consisting of multiple categories of electronic waste. Images are resized, normalized, and split into training and validation datasets before training the deep learning model.

## 🧠 Workflow

1. Import required libraries
2. Load the E-Waste image dataset
3. Preprocess and resize images
4. Normalize image data
5. Split data into training and validation sets
6. Build the CNN model
7. Train the model
8. Evaluate model accuracy
9. Predict classes for new images
